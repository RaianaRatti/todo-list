# To-Do List

A simple browser-based to-do list built with plain HTML, CSS, and JavaScript. Add tasks with a due date and delete them when they're done.

## Project in Action

🎥 Watch the demo video!

## Features

- Add a task with a name and a due date
- Add a task by pressing Enter or clicking the Add button
- Use an in-built calender to add the date
- Delete any task from the list
- No dependencies, runs directly in the browser

## Files

- `todo.html` - page structure
- `todo.css` - styling
- `todo.js` - task logic (add, delete, render)

## Usage

Open `todo.html` in any web browser. Type a task name, pick a date, and click Add (or press Enter) to add it to the list. Click Delete next to a task to remove it.
